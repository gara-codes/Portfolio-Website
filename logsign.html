LogIn-SignUp app

A few weeks after learning Android Studio, we were taught how to write a basic REST api which uses HTTP to query the backend and interact with the frontend using basic php scripts. A simple project that allows a user to sign up using an email and password and then log in using those same credentials. There are various data validation checks such as if the user already exits, if the user does not exist or if the password is incorrect.
